## About the project
It is a simple application built to show the top 10 open-source projects with the most number of stars and forks in the last 24 hours.
It has been built using [OpenSauced API](https://api.opensauced.pizza/). 

## Built using:
- OpenSauced API
- TailwindCSS
- JavaScript
